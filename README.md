This Power BI dashboard is designed to drive healthcare efficiency by identifying performance outliers and uncovering the key attributes
influencing patient costs and length of stay. By leveraging advanced DAX modeling and exploratory analytics, the project transforms raw 
clinical data into a cohesive, interactive tool tailored for the HealthStat brand. The final report integrates complex calculated measures 
with user-friendly navigation to provide actionable insights into hospital operations and financial performance.
